#### 功能
- 推薦行程
- 旅行日記
- 關於我們
- 常見問題
- 購物車

#### 專案技術
- Vue 3.2.45
- Vite 4.0.0
- Pinia 2.0.28
- Vue Axios 3.5.2
- Vue Router 4.1.6
- Tailwind CSS 3.2.7
- Vee-validate 4.7.4
- Vue Loading Overlay 6.0.2
- Animate.css 4.1.1
- Swiper 9.1.0 
- SweetAlert2 11.7.3
- Sass

#### 安裝套件
```sh
npm install
```
#### 運行專案
```sh
npm run dev
```

#### 開啟專案
在瀏覽器網址列輸入以下即可看到畫面
```sh
http://localhost:3000/
```



#### 打包專案
```sh
npm run build
```

#### 部署專案
```sh
sh deploy.sh
```
